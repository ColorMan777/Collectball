# CollectBall

A tiny game made in 2 days with Unity 6.2.

### Controls :
WASD / ZQSD + Space + Mouse

### Features :
Rigid body physics (built-in Unity) / Level changer / Sound and 3D Sound / Collectibles System / Particles / Dynamic Keys Counting to end the level / Moving Platforms / Menus / and what is left of my sleep schedule ...

Have fun !

*Note* *: you can also get the game on itch.io : https://colorman777.itch.io/collectball*
