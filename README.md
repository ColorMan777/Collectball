# CollectBall

A tiny game made in 2 days.

Controls : WASD / ZQSD + Space + Mouse

Features : Rigid body physics (built-in Unity) / Level changer / Sound and 3D Sound / Collectibles System / Particles / Dynamic Keys Counting to end the level / Moving Platforms / Menus / and what is left of my sleep schedule ...

Have fun !
